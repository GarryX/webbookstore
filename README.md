# webbookstore
A project for bookstore!

    该项目利用Maven工具及SSH（Structs2, Spring4 及 Hibernate4）框架,实现分页，跳页，条件查询，
查看商添加商品到购物车，利用Ajax技术修改购物车中商品的数量并返回新的总账单额，删除
购物车中的购物项，清空购物车，结账，用户名账号验证，余额验证，库存验证，并最终实现
模拟结账的事务操作。另外，本项目也有额外的用户购物记录查询功能，可以通过用户名检索
出该用户在本网上书城的购物历史。
    项目总体侧重功能实现，在表现层略显粗糙。是一个基本的模拟网上书城项目。有待改进
的地方还有很多。谢谢。
